1. Đăng nhập hệ thống Authentication

sequenceDiagram autonumber actor User as 👤 User participant LP as 🖥️ Login Page participant MW as 🔒 Middleware participant Auth as 🔐 Supabase Auth participant DB as 🗄️ Database participant Dashboard as 📊 Dashboard participant Profile as 👤 Profile User->>LP: Nhập email & password LP->>Auth: signInWithPassword() Auth->>Auth: Validate credentials alt Đăng nhập thành công Auth-->>LP: Session + User data LP->>MW: Redirect với session MW->>DB: Query employee role DB-->>MW: Role (ADMIN/MANAGER/EMPLOYEE) alt Role = ADMIN hoặc MANAGER MW->>Dashboard: Redirect /dashboard Dashboard-->>User: Hiển thị Dashboard else Role = EMPLOYEE MW->>Profile: Redirect /profile Profile-->>User: Hiển thị Profile cá nhân end else Đăng nhập thất bại Auth-->>LP: Error message LP-->>User: Hiển thị lỗi end

Mô tả luồng

  1. User nhập thông tin đăng nhập trên Login Page
  2. Hệ thống gọi Supabase Auth để xác thực
  3. Middleware kiểm tra role của user từ database
  4. Dựa vào role, redirect đến trang phù hợp (Dashboard hoặc Profile)

2. Quản lý Nhân viên (CRUD) Employee Management

sequenceDiagram autonumber actor Admin as 👑 Admin/Manager participant Page as 🖥️ Employees Page participant Action as ⚡ Server Action participant Service as 🔧 EmployeeService participant Repo as 📦 EmployeeRepo participant DB as 🗄️ Supabase DB Note over Admin, DB: === Xem danh sách === Admin->>Page: Truy cập /employees Page->>Service: getEmployeesList() Service->>Repo: getEmployees() Repo->>DB: SELECT * FROM employees DB-->>Repo: Employees data Repo-->>Service: Employee[] Service-->>Page: Employee[] Page-->>Admin: Hiển thị danh sách Note over Admin, DB: === Thêm mới === Admin->>Page: Click "Thêm nhân viên" Page->>Page: Mở form thêm mới Admin->>Page: Điền thông tin & Submit Page->>Action: createEmployee(formData) Action->>Service: createEmployee(formData) Service->>Service: requireManagerOrAbove() Service->>Service: Validate data Service->>Repo: createEmployee(data) Repo->>DB: INSERT INTO employees DB-->>Repo: New employee Repo-->>Service: Employee Service-->>Action: Success Action-->>Page: Revalidate & Redirect Page-->>Admin: Danh sách cập nhật Note over Admin, DB: === Cập nhật === Admin->>Page: Click "Sửa" nhân viên Page->>Service: getEmployee(id) Service-->>Page: Employee data Page-->>Admin: Hiển thị form edit Admin->>Page: Chỉnh sửa & Submit Page->>Action: updateEmployee(id, formData) Action->>Service: updateEmployee(id, formData) Service->>Repo: updateEmployee(id, data) Repo->>DB: UPDATE employees SET... DB-->>Repo: Updated employee Repo-->>Page: Success Page-->>Admin: Thông báo thành công Note over Admin, DB: === Xóa === Admin->>Page: Click "Xóa" nhân viên Page->>Page: Confirm dialog Admin->>Page: Xác nhận xóa Page->>Action: deleteEmployee(id) Action->>Service: deleteEmployee(id) Service->>Repo: deleteEmployee(id) Repo->>DB: DELETE FROM employees DB-->>Service: Success Service->>Auth: Delete Auth User (nếu có) Service-->>Page: Success Page-->>Admin: Danh sách cập nhật

Mô tả luồng

  1. View: Page gọi Service → Repository → Database để lấy dữ liệu
  2. Create: Form submit → Server Action → Service (validate + auth check) → Repo → DB
  3. Update: Fetch data → Edit form → Submit → Service → Repo → DB
  4. Delete: Confirm → Service xóa employee và Auth user liên quan

3. Chấm công (Check-in/Check-out) Attendance

sequenceDiagram autonumber actor EMP as 👤 Employee participant Page as 🖥️ Calendar Page participant Service as 🔧 AttendanceService participant Repo as 📦 AttendanceRepo participant DB as 🗄️ Supabase DB Note over EMP, DB: === Check-in === EMP->>Page: Click "Check-in" Page->>Service: checkIn(employeeId) Service->>Service: Get current time Service->>Repo: findTodayAttendance(empId) Repo->>DB: SELECT WHERE date = today alt Chưa có record hôm nay DB-->>Repo: null Repo->>DB: INSERT (employee_id, date, check_in_time) DB-->>Repo: New attendance else Đã có record (chưa check-in) DB-->>Repo: Attendance record Repo->>DB: UPDATE SET check_in_time = now() DB-->>Repo: Updated attendance end Repo-->>Service: Attendance Service-->>Page: Success + Time Page-->>EMP: Hiển thị giờ check-in Note over EMP, DB: === Check-out === EMP->>Page: Click "Check-out" Page->>Service: checkOut(employeeId) Service->>Repo: findTodayAttendance(empId) Repo->>DB: SELECT WHERE date = today DB-->>Repo: Attendance (with check_in) alt Đã check-in Repo->>DB: UPDATE SET check_out_time = now() DB-->>Repo: Updated attendance Repo-->>Service: Success Service-->>Page: Success + Time Page-->>EMP: Hiển thị giờ check-out else Chưa check-in Repo-->>Service: Error Service-->>Page: Error message Page-->>EMP: "Vui lòng check-in trước" end

Mô tả luồng

  1. Check-in: Tạo hoặc cập nhật record chấm công với thời gian hiện tại
  2. Check-out: Chỉ cho phép nếu đã check-in, cập nhật check_out_time
  3. Hệ thống tự động xác định trạng thái (Present, Late, Absent...)

5. Tính lương tháng Payroll

sequenceDiagram autonumber actor Admin as 👑 Admin participant Page as 🖥️ Payroll Page participant Service as 🔧 PayrollService participant EmpRepo as 📦 EmployeeRepo participant AttRepo as 📦 AttendanceRepo participant OTRepo as 📦 OvertimeRepo participant LeaveRepo as 📦 LeaveRepo participant PayRepo as 📦 PayrollRepo participant DB as 🗄️ Database Admin->>Page: Chọn tháng/năm & Click "Tính lương" Page->>Service: generateMonthlyPayroll(month, year) Service->>EmpRepo: getEmployees() EmpRepo->>DB: SELECT * FROM employees DB-->>Service: Employee[] loop Với mỗi Employee Service->>AttRepo: getAttendanceByEmployee(empId, month) AttRepo-->>Service: Attendance[] (work days) Service->>OTRepo: getApprovedOTByEmployee(empId, month) OTRepo-->>Service: OvertimeRequest[] (OT hours) Service->>LeaveRepo: getApprovedLeavesByEmployee(empId, month) LeaveRepo-->>Service: LeaveRequest[] (leave days) Service->>Service: Calculate base salary Service->>Service: Calculate OT salary Service->>Service: Calculate bonus/penalties Service->>Service: Calculate tax (progressive) Service->>Service: Calculate social insurance Service->>Service: Calculate net_pay Service->>PayRepo: createOrUpdatePayslip(data) PayRepo->>DB: UPSERT payslips end DB-->>Service: All payslips created Service-->>Page: Payslip[] Page-->>Admin: Hiển thị bảng lương Note over Admin, DB: === Thanh toán === Admin->>Page: Click "Đánh dấu đã thanh toán" Page->>Service: markPayrollAsPaid(month, year) Service->>PayRepo: updateStatus(Paid) PayRepo->>DB: UPDATE payslips SET status = 'Paid' DB-->>Page: Success Page-->>Admin: "Đã thanh toán"

Công thức tính lương

  1. Lương cơ bản: base_salary × (work_days / total_work_days)
  2. Lương OT: (base_salary / 26 / 8) × 1.5 × ot_hours
  3. Thuế TNCN: Progressive tax calculation (5% - 35%)
  4. BHXH: 10.5% × base_salary
  5. Net Pay: Base + OT + Bonus - Tax - BHXH - Penalties - Advance
