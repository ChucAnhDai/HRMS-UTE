3. Feature Flow: Monthly Payroll Calculation

Sequence showing how Overtime data is integrated into Payroll generation.
sequenceDiagram
    participant Admin
    participant PayrollService
    participant EmployeeRepo
    participant AttendanceRepo
    participant OvertimeRepo
    participant PayrollRepo
    participant Database

    Admin->>PayrollService: generateMonthlyPayroll(Month, Year)
    PayrollService->>EmployeeRepo: getActiveEmployees()
    EmployeeRepo-->>PayrollService: List[Employees]

    loop For Each Employee
        PayrollService->>AttendanceRepo: getMonthlyStats(empId)
        AttendanceRepo-->>PayrollService: WorkDays, LateDays

        PayrollService->>OvertimeRepo: getMonthlyApprovedHours(empId)
        OvertimeRepo-->>PayrollService: Total OT Hours

        Note over PayrollService: Calculate Salary:
Base + (OT * 1.5) - Deductions

        PayrollService->>PayrollService: Create Payslip Object
    end

    PayrollService->>PayrollRepo: createPayslips(List[Payslips])
    PayrollRepo->>Database: INSERT INTO payslips
    Database-->>PayrollRepo: Success
    PayrollRepo-->>PayrollService: Success
    PayrollService-->>Admin: Done
